Wearable Ankle Motion Tracking System

Embedded wearable sensing prototype that measures ankle motion using an MPU-6050 and ESP32, processes motion data in real time, and streams live results to a browser dashboard over Wi-Fi.

← Back to Portfolio

Project Overview

This project was built as an embedded biomechanics prototype for ankle motion tracking. The system uses an MPU-6050 inertial measurement unit and an ESP32 microcontroller to capture motion data, estimate ankle movement, detect motion events, and display live results in a browser-based dashboard.

The goal was to build more than a basic sensor demo. The project was structured as a complete embedded system with hardware integration, real-time signal processing, Wi-Fi communication, and live visualization.

What the System Does

  • Reads 3-axis acceleration and gyroscope data from the MPU-6050
  • Uses I2C communication between the ESP32 and the sensor
  • Performs angle estimation from inertial sensor readings
  • Implements threshold-based motion detection logic
  • Tracks motion events, step count, and cadence
  • Hosts a browser-accessible dashboard through the ESP32 over Wi-Fi
  • Displays live motion values and plots in real time

Engineering Work Completed

  • Integrated the MPU-6050 with the ESP32 at the hardware and firmware level
  • Built embedded code for sensor readout and motion data handling
  • Implemented real-time calculations for angle and motion state
  • Created browser-based data output from the ESP32
  • Tested the system as a working breadboard prototype

Why This Project Matters

This project demonstrates embedded systems development beyond simple output control. It combines sensor interfacing, real-time data processing, motion analysis, and user-facing visualization into a single system.

It also creates a strong foundation for future work in wearable sensing, biomechanics, robotics, feedback systems, and real-time monitoring applications.
